# Open-Ended Causal POET

## Summary

A research-oriented implementation of POET-style open-ended causal task generation with novelty, solvability, genealogy, and transfer analysis.

This repository is organized as a research-style prototype rather than a casual demo.  
It includes method code, experiment scripts, tests, result artifacts, documentation, and a Streamlit viewer.

## Research orientation

The project is intended to support research planning in:

- artificial life,
- causal world models,
- open-ended discovery,
- multi-agent systems,
- automated scientific discovery.

## Evidence generated by the repository

The repository produces:

- CSV logs,
- quantitative metrics,
- visual diagnostics,
- model comparison plots,
- animations where relevant.

## How to read the repository

1. Start with `README.md`.
2. Run `python -m pytest -q`.
3. Run `python scripts/run_all.py`.
4. Inspect `results/`.
5. Open `streamlit run app.py`.
